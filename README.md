# Calculator Project

## Description

This project is a web-based calculator that allows users to perform basic arithmetic operations. It features a modern design with an animated background for an enhanced visual experience.

## Features

- **Basic Operations**: Addition, Subtraction, Multiplication, Division, and Percentage.
- **Clear and Delete**: Clear all input (AC) and delete the last character (DEL).
- **Responsive Design**: Adjusts to different screen sizes for better usability on various devices.
- **Animated Background**: Visually appealing background with animated bubbles.

## Technologies Used

- **HTML**: Provides the structure of the calculator.
- **CSS**: Styles the calculator and adds animations.
- **JavaScript**: Implements the calculator's functionality.

## Getting Started

### Prerequisites

To run this project, you need a web browser.

### Installation

1. **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/calculator-project.git
    ```

2. **Navigate to the project directory**:
    ```bash
    cd calculator-project
    ```

3. **Open `index.html` in your web browser** to view and use the calculator.
